Posted on 2017-02-09 Edited on 2019-04-17 In shell Comments:
Linux命令行查看 CPU 信息。

本文记录 Linux 操作系统中查看 CPU 信息的方法。查看的内容包含但不局限于：

  • CPU 型号
  • CPU 核数
  • CPU 主频
  • Cache 容量
  • CPU 指令集
  • CPU 特性

cat /proc/cpuinfo

dmesg | grep Hz

lscpu

lshw -c cpu

sudo dmidecode -t processor

sudo watch -n 1 cat /sys/devices/system/cpu/cpu*/cpufreq/cpuinfo_cur_freq

监控 CPU 频率变化。
